## Run Locally

Clone the project

```bash
  git clone https://github.com/mkoroleva5/rs_GraphiQL_front.git
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
